Aula 14: Funções e Parâmetros

Funções: Criando Nossas Próprias Ferramentas

1. O Problema: Código Repetido

Imagine que você precisa calcular a média de notas em várias partes do seu programa. Você teria que escrever o mesmo código de soma e divisão várias vezes. Isso é ruim porque:

  • Dá mais trabalho.
  • Aumenta a chance de erros (você pode errar em uma das cópias).
  • Dificulta a manutenção (se precisar mudar a lógica, tem que mudar em todos os lugares).

2. O que é uma Função?

Uma função é um bloco de código que realiza uma tarefa específica e que pode ser reutilizado. Pense nela como uma "máquina" que você constrói uma vez e pode usar quantas vezes quiser.

Funções nos ajudam a seguir o princípio DRY (Don't Repeat Yourself - Não se Repita), um dos conceitos mais importantes da programação.

3. Criando uma Função Simples

A forma mais básica de uma função é aquela que apenas executa uma ação, sem receber ou devolver informações.

// Definindo a função funcao mostrarMenu() { escreva("--- MENU ---\n") escreva("1 - Jogar\n") escreva("2 - Sair\n") } // Usando (ou "chamando") a função dentro do início funcao inicio() { mostrarMenu() }

4. Parâmetros: Os "Ingredientes" da Função

Para uma função ser realmente útil, muitas vezes ela precisa de informações para trabalhar. Parâmetros são as variáveis que a função recebe para poder executar sua tarefa.

// Esta função recebe um 'nome' como parâmetro funcao saudar(cadeia nome) { escreva("Olá, ", nome, "! Bem-vindo(a).") } funcao inicio() { saudar("Vitor") // A função é chamada com o argumento "Vitor" saudar("Maria") // Reutilizando a mesma função com outro argumento }

5. Retorno: O "Produto Final" da Função

Além de executar ações, uma função também pode calcular um valor e retorná-lo para quem a chamou. Para isso, precisamos definir o tipo de dado que a função vai retornar e usar a palavra `retorne`.

// Esta função recebe dois inteiros e retorna um inteiro funcao inteiro somar(inteiro n1, inteiro n2) { inteiro resultado = n1 + n2 retorne resultado } funcao inicio() { inteiro somaTotal = somar(10, 5) // O valor retornado (15) é guardado na variável escreva("O resultado da soma é: ", somaTotal) // Saída: 15 }
