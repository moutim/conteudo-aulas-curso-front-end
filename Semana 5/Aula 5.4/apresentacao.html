Aula 13: O Laço 'enquanto' (while)

O Laço `enquanto` (while)

1. Repetindo sem Saber o Fim

O laço `para` (for) é ótimo quando sabemos exatamente quantas vezes queremos repetir algo (ex: percorrer as 5 posições de um vetor). Mas e se precisarmos repetir uma ação até que o usuário decida parar?

O laço enquanto (while) resolve isso. Ele executa um bloco de código repetidamente enquanto uma determinada condição for verdadeira.

2. A Sintaxe do `enquanto`

A estrutura é simples: o laço testa a condição. Se for verdadeira, ele executa o bloco de código e depois testa a condição novamente. Ele só para quando a condição se torna falsa.

// A variável de controle é declarada FORA do laço inteiro contador = 0 enquanto (contador < 5) { escreva("O valor do contador é: ", contador) // É crucial atualizar a variável de controle DENTRO do laço contador = contador + 1 } // Saída: // O valor do contador é: 0 // O valor do contador é: 1 // O valor do contador é: 2 // O valor do contador é: 3 // O valor do contador é: 4

3. `para` vs. `enquanto`: Qual Usar?

Use `para` (for)

Quando você tem um número definido de repetições.

  • Percorrer um vetor de tamanho fixo.
  • Executar algo exatamente 10 vezes.
  • Fazer uma contagem regressiva.

Use `enquanto` (while)

Quando a condição de parada é indeterminada e depende de um evento.

  • Esperar o usuário digitar "sair".
  • Validar uma senha.
  • Manter um menu de opções ativo.

4. Lidando com Menus: `escolha caso` (switch case)

Quando temos um menu com várias opções, usar uma cadeia longa de se ... senao se ... senao pode ficar confuso. Para esses casos, usamos a estrutura escolha caso.

Ela avalia uma única variável e executa o bloco de código correspondente ao valor dela. É perfeito para menus!

inteiro opcao escreva("Escolha uma opção (1 ou 2): ") leia(opcao) escolha (opcao) { caso 1: escreva("Você escolheu a opção 1.") pare // O 'pare' é essencial para sair da estrutura caso 2: escreva("Você escolheu a opção 2.") pare caso contrario: // Executado se nenhuma das opções acima for escolhida escreva("Opção inválida!") }

5. O Perigo do Loop Infinito

Se a condição do laço `enquanto` nunca se tornar falsa, o programa ficará preso executando o mesmo código para sempre. Isso é um loop infinito e fará seu programa travar.

// PERIGO: Exemplo de loop infinito inteiro contador = 0 enquanto (contador < 5) { escreva("Isso vai repetir para sempre!") // O erro: esquecemos de incrementar o contador. // Ele sempre será 0, e 0 é sempre menor que 5. }
