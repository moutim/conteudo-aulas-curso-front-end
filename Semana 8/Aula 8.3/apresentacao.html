2. CSS Externo (CDN) e `Animate.css`

Não precisamos escrever todas as nossas animações ou estilos do zero. Podemos usar bibliotecas de CSS prontas, como a Animate.css.

O que é um CDN?

CDN (Content Delivery Network) é um serviço que hospeda arquivos (como bibliotecas CSS ou JS) em servidores ao redor do mundo. Em vez de baixar o arquivo, nós apenas o linkamos no nosso ``.

Usando Animate.css

É uma biblioteca com dezenas de animações prontas que podemos usar apenas adicionando classes ao nosso HTML.

<!-- 1. Adicione o link do CDN no <head> --> <head> <link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/animate.css/4.1.1/animate.min.css" /> </head> <!-- 2. Use as classes no HTML para animar --> <h1 class="animate__animated animate__bounce">Isso aqui pula!</h1> <!-- 3. Use com JS para animar em um evento --> <button id="meu-botao">Clique para animar</button> <p id="texto-animado">Eu vou aparecer com fade!</p> <script> const botao = document.getElementById('meu-botao'); const texto = document.getElementById('texto-animado'); botao.addEventListener('click', () => { texto.classList.add('animate__animated', 'animate__fadeInUp'); }); </script>

3. Copiando Texto (Clipboard)

Muitas vezes queremos dar ao usuário um botão para "Copiar" um código, um link de convite ou um cupom.

Método `navigator.clipboard` (Moderno)

A forma mais nova e limpa de fazer isso. Funciona com `async/await` e é baseada em "Promessas". Nota: Pode não funcionar em todos os ambientes (ex: alguns iframes ou sites não-HTTPS).

async function copiarTextoModerno(texto) { try { await navigator.clipboard.writeText(texto); alert('Texto copiado com sucesso!'); } catch (err) { console.error('Falha ao copiar: ', err); } } // Para usar: // copiarTextoModerno('Seu cupom: VITOR10');

Método `document.execCommand` (Compatível)

É a forma mais antiga, mas funciona em praticamente todos os lugares, incluindo ambientes de teste. Ela é um pouco mais "manual":

function copiarTextoCompativel(texto) { // 1. Cria um textarea temporário fora da tela const textarea = document.createElement('textarea'); textarea.value = texto; document.body.appendChild(textarea); // 2. Seleciona o texto dentro do textarea textarea.select(); // 3. Executa o comando "copiar" try { document.execCommand('copy'); alert('Texto copiado!'); } catch (err) { console.error('Falha ao copiar', err); } // 4. Remove o textarea temporário document.body.removeChild(textarea); } // Para usar: // copiarTextoCompativel('Seu cupom: VITOR10');
