Aula 16: JavaScript Interagindo com HTML

Manipulando o DOM com JavaScript

1. O que é o DOM?

Quando o navegador carrega uma página HTML, ele cria uma representação da estrutura dessa página na memória. Essa representação é chamada de DOM (Document Object Model).

Pense no DOM como uma árvore hierárquica onde cada tag HTML é um "nó" ou "objeto". O JavaScript pode acessar e modificar essa árvore, alterando assim o que é exibido na página sem precisar recarregá-la.

Exemplo de HTML:

<html> <head> <title>Minha Página</title> </head> <body> <h1>Título</h1> <p>Um parágrafo.</p> </body> </html>

Árvore DOM Correspondente:

Document
└── html
    ├── head
    │   └── title
    │       └── #text ("Minha Página")
    └── body
        ├── h1
        │   └── #text ("Título")
        └── p
            └── #text ("Um parágrafo.")

2. Selecionando Elementos HTML

Para modificar um elemento, primeiro precisamos "encontrá-lo" no DOM. JavaScript nos oferece várias formas de fazer isso.

`getElementById()`: Buscando por ID

A forma mais direta. Se um elemento HTML tem um atributo `id` único, podemos selecioná-lo facilmente.

// HTML:

Olá

const titulo = document.getElementById('titulo-principal'); // Agora a variável 'titulo' representa o elemento H1

`querySelector()`: Buscando com Seletor CSS

Mais versátil. Permite usar os mesmos seletores que usamos no CSS para encontrar o primeiro elemento que corresponda.

// HTML: const botao = document.querySelector('.botao-confirmar'); // Busca pela classe const primeiroParagrafo = document.querySelector('p'); // Busca pela tag const elementoComID = document.querySelector('#id-especifico'); // Busca pelo ID

`querySelectorAll()`: Buscando Vários Elementos

Similar ao `querySelector`, mas retorna uma lista (NodeList) de todos os elementos que correspondem ao seletor.

// HTML:
  • Item 1
  • Item 2
    • const todosOsItensDaLista = document.querySelectorAll('li'); // Retorna uma lista com os dois LIs

    3. Modificando o Conteúdo

    Uma vez que selecionamos um elemento, podemos alterar seu conteúdo de texto ou até mesmo o HTML dentro dele.

    `innerText` ou `textContent`: Alterando o Texto

    Ambos servem para ler ou definir o conteúdo de texto de um elemento, ignorando as tags HTML.

    const titulo = document.getElementById('titulo-principal'); titulo.innerText = "Novo Título da Página"; // Muda o texto do H1 console.log(titulo.textContent); // Mostra o texto atual no console

    `innerHTML`: Alterando o HTML Interno

    Permite ler ou definir todo o conteúdo HTML dentro de um elemento. Use com cuidado, pois pode abrir brechas de segurança se usado com dados do usuário.

    const caixa = document.querySelector('.minha-caixa'); caixa.innerHTML = "<strong>Texto em negrito</strong> dentro da caixa.";

    4. Modificando Estilos CSS

    Podemos alterar estilos diretamente ou, de forma mais organizada, manipulando as classes CSS do elemento.

    Diretamente com `style`

    Útil para alterações rápidas ou dinâmicas. Lembre-se que propriedades CSS com hífen viram camelCase.

    const botao = document.querySelector('button'); botao.style.backgroundColor = '#e80170'; botao.style.color = 'white';

    Manipulando Classes com `classList` (Recomendado)

    É a forma mais limpa e organizada de gerenciar estilos complexos. Você define as classes no seu CSS e usa JavaScript para adicioná-las ou removê-las.

    // CSS: .ativo { background-color: #e80170; color: white; } const itemMenu = document.querySelector('#menu-item-1'); itemMenu.classList.add('ativo'); // Adiciona a classe itemMenu.classList.remove('ativo'); // Remove a classe // O mais útil: toggle! // Se a classe existe, remove. Se não existe, adiciona. Perfeito para botões e menus. itemMenu.classList.toggle('ativo');

    5. Respondendo a Ações: Event Listeners

    A parte mais legal é fazer a página responder às ações do usuário (cliques, digitação, etc.). Usamos Event Listeners para "ouvir" esses eventos e executar uma função quando eles acontecem.

    // HTML: //

    const botao = document.getElementById('meu-botao'); const paragrafoMensagem = document.getElementById('mensagem'); // Adiciona um "ouvinte" para o evento de 'click' no botão botao.addEventListener('click', function() { // Esta função será executada QUANDO o botão for clicado paragrafoMensagem.textContent = "O botão foi clicado!"; botao.classList.toggle('ativo'); // Exemplo usando toggle no clique });

    Existem muitos tipos de eventos: click, mouseover (passar o mouse), keydown (pressionar tecla), submit (enviar formulário), etc. Veja uma lista completa aqui.

    6. Conclusão

    Dominar a manipulação do DOM é a chave para criar páginas web dinâmicas e interativas. Hoje vimos o essencial:

    • Selecionar elementos (getElementById, querySelector).
    • Alterar conteúdo (innerText, innerHTML).
    • Alterar estilos (.style e .classList).
    • Ouvir eventos (addEventListener).

    Com essas ferramentas, vocês já podem começar a criar pequenas interações nas suas páginas!
