Aula 15: Introdução ao JavaScript

JavaScript: A Linguagem da Web Interativa

1. O que é JavaScript?

Até agora, aprendemos a criar a estrutura de uma página com HTML e a estilizá-la com CSS. O JavaScript (JS) é a terceira camada: a linguagem de programação que nos permite adicionar interatividade, lógica e dinamismo às nossas páginas.

Se o HTML é o esqueleto e o CSS é a pele, o JavaScript é o sistema nervoso e os músculos.

2. Onde o JavaScript Roda?

No Navegador (Front-end)

É o seu ambiente natural. Todo navegador moderno tem um "motor" JavaScript (como o V8 do Chrome) que executa o código e permite manipular a página, responder a cliques, validar formulários, etc.

Fora do Navegador (com Node.js)

Graças ao Node.js, podemos usar JavaScript para construir o back-end de aplicações, criar ferramentas de linha de comando e muito mais, rodando diretamente na nossa máquina.

3. Como Adicionar JavaScript a uma Página

Existem duas maneiras principais de incluir código JavaScript em um arquivo HTML.

Método 1: Script Interno

Podemos escrever o código diretamente no HTML, dentro da tag <script>. Geralmente, colocamos essa tag no final do <body>.

<body> <!-- Conteúdo do site --> <script> console.log("Olá, Mundo! Isso roda no console do navegador."); </script> </body>

Método 2: Script Externo (Recomendado)

A melhor prática é criar um arquivo separado com a extensão .js e linká-lo ao nosso HTML. Isso mantém o código organizado.

<!-- No arquivo index.html --> <body> <!-- Conteúdo do site --> <script src="meu_script.js"></script> </body> <!-- No arquivo meu_script.js --> console.log("Este código veio de um arquivo externo!");

4. Declarando Variáveis: `var`, `let` e `const`

Em JavaScript moderno, usamos principalmente `let` e `const` para declarar variáveis. A palavra `var` é mais antiga e deve ser evitada.

`const` (Constante)

Use para valores que não vão mudar. Uma vez declarada, você não pode atribuir um novo valor a ela. É a sua escolha padrão.

const nome = "Vitor";

`let` (Variável)

Use para valores que precisam ser alterados ao longo do tempo, como um contador em um laço.

let idade = 25; idade = 26; // Funciona!

`var` (Antigo)

A forma antiga de declarar variáveis. Tem um comportamento de "escopo" que pode causar bugs. Evite usar em código novo.

Regra de Ouro

Comece sempre declarando suas variáveis com const. Se depois você perceber que precisa mudar o valor dela, troque para let.
